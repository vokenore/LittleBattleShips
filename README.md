# Морской бой

Консольная игра на Java с поддержкой мультиплеера.

## Запуск

Windows: `run.bat`

Linux / Mac: `bash run.sh`

Требования: Java 8+

---

## Начало игры

При запуске введите имя. Доступные режимы:
- `1` - Одиночная игра против бота
- `2` - Игра по сети (WebSocket)

Имя `admin` открывает режим администратора.

---

## Игровое поле

Поле 16×16. Колонки обозначены буквами A-P, строки цифрами 1-16.

| Символ | Значение |
|--------|----------|
| `.` | Пустая клетка |
| `O` | Корабль (только на своём поле) |
| `X` | Попадание |
| `*` | Промах |

---

## Состав флота

| Размер | Количество |
|--------|-----------|
| 6 клеток | 1 шт |
| 5 клеток | 2 шт |
| 4 клетки | 3 шт |
| 3 клетки | 4 шт |
| 2 клетки | 5 шт |
| 1 клетка | 6 шт |

---

## Расстановка кораблей

Формат команды: `[колонка][строка] [H/V]`

Примеры:

- `A1 H` - корабль нужного в момент игры размера, горизонтально от A1 
- `P16 V` - корабль размером 2, вертикально от P16
- `auto` - разместить текущий корабль автоматически
- `allauto` - разместить все корабли автоматически

---

## Ход игры

Вводи координату клетки для выстрела, например: `A5`, `H12`, `P1`

- **Ранил** или **Убил** - стреляешь снова
- **Мимо** - ход переходит противнику

---

## Игра по сети (WebSocket)

**Первый игрок** выбирает режим `2 -> 1`, вводит порт (по умолчанию 8887) и сообщает второму игроку свой IP и порт.

**Второй игрок** выбирает режим `2 -> 2`, вводит IP первого игрока и порт.

**Наблюдатель** — если оба места заняты, следующий подключившийся автоматически получает роль наблюдателя и видит все ходы в реальном времени.

---

## Режим администратора

Введи имя `admin` при запуске. Отображается список всех сыгранных игр.

| Команда | Описание |
|---------|----------|
| `view N` | Просмотр игры с воспроизведением ходов по Enter |
| `delete N` | Удалить игру |
| `archive N` | Архивировать игру в zip |
| `exit` | Выйти |

В режиме `view N` нажимай Enter для воспроизведения каждого хода - поля обоих игроков обновляются на экране.

---

## Логи игр

После каждой завершённой игры создаётся файл в папке `logs/`, например `logs/game_2024-01-15_14-30.txt`.

Файл содержит: время старта и финиша, победителя, статистику каждого игрока (ходы, попадания, точность %), финальные поля, пошаговый список всех ходов.

---

## Алгоритм бота

- **Поиск** - стреляет по шахматному паттерну, гарантируя обнаружение любого корабля размером 2+
- **Добивание** - после попадания добивает корабль, определяя направление по второму попаданию

Бот принимает решения только на основе результатов своих выстрелов.
